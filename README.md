provide a convenient solution for mobile users to get log messages from ex. console.log ----- 在移动平台下即使不连接电脑也可以方便获取日志信息。

---

## Usage

```bash
$ npm i dora dora-plugin-debug-corner -SD
$ ./node_modules/.bin/dora --plugins debug-corner
```

## Test

```bash
$ npm test
```
